# tools/binary — Binary Analysis Tools

> Tools for inspecting and analyzing binary files.

Includes:
- `uelf` — ELF inspector tailored for Tattva binaries
- `udis` — disassembler (x86-64, AArch64, RISC-V)
- `uhex` — hex dump with assembly annotation
- `usect` — ELF section manipulator
- `usize` — binary size analyzer

Part of the `tools/` category in Tattva OS.
