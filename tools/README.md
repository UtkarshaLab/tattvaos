# tools — Developer Tools Category

> Developer tooling for Tattva OS.
> The full ecosystem of tools that make working on Tattva OS possible.

---

## Projects

| Project | Description |
|---|---|
| [`devtools/`](devtools/) | Core dev tools — debugger, formatter, REPL, step executor |
| [`binary/`](binary/) | Binary analysis tools — ELF inspector, disassembler, hex dump |
| [`coretools/`](coretools/) | Core system tools — shell (ish), text editor (Kalam), dashboard |
| [`unix/`](unix/) | POSIX-compatible layer — familiar tools reimplemented natively |

---

## Key Tools

- `utasm` — assembler/linker (in `asm/`)
- `ish` — Ishana shell
- `Kalam` — text editor
- `udbg` — native debugger, no DWARF dependency
- `usym` — symbol table inspector
- `udis` — disassembler
- `umap` — memory map visualizer
