# tools/unix — POSIX-Compatible Utilities

> Familiar UNIX utilities reimplemented natively for Tattva OS.

Provides a POSIX-compatible interface for developers accustomed to Linux tools.
All implementations are native assembly — no GNU dependency.

Includes: ls, cat, cp, mv, rm, mkdir, find, grep, sed, sort, wc, head, tail.

Part of the `tools/` category in Tattva OS.
