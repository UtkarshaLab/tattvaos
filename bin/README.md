# bin — Compiled Binaries

> Output directory for compiled Tattva OS binaries.
> Not tracked in version control — generated by the build system.

---

All binaries here are produced by `make` or `utasm`.
Do not commit binaries. The `.gitignore` excludes this directory's contents.

Build outputs:
- `tattva.img` — bootable disk image
- `kernel.bin` — kernel flat binary
- `*.upk` — UPM package files
