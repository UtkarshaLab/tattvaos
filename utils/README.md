# utils — System Utilities Category

> Native system utilities for Tattva OS.
> File, process, text, and system management tools.

---

## Projects

| Project | Description |
|---|---|
| [`file/`](file/) | File utilities — copy, move, delete, stat, find |
| [`dir/`](dir/) | Directory utilities — list, create, navigate, tree |
| [`text/`](text/) | Text processing utilities — grep, sed, awk equivalents |
| [`process/`](process/) | Process utilities — list, inspect, signal, kill |
| [`system/`](system/) | System utilities — hardware info, uptime, environment |
| [`archive/`](archive/) | Archive utilities — pack, unpack, compress |
| [`docs/`](docs/) | Utility documentation |

---

## Philosophy

These are the POSIX-style utilities rewritten for Tattva OS.
Familiar interface. Native performance. No GNU dependency.
